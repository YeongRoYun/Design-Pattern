## Environments
**Language**) JAVA 17  
**Build**) Gradle  
**IDE**) Intellij  
**Test Tool**) JUnit5  

## Setup
1. Clone this repository into a local machine.
2. Open it with Intellij.
3. Run all test cases directly by `gradle test`
4. Run a test case directly by Intellij UI
5. Add JUnit at **Run Configuration**
